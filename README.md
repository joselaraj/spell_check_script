# Spelling and capitalization checker
Reads a text file, checks for spelling mistakes, and flags sentences that don't start with a capital letter.
## Requirements
- Python
- pyspellschecker library
  - How to install:
    pip install pyspellchecker
## How to run 
- For Linux/Macos: python3 main.py [yourfile.txt]
- For Windows: python main.py [yourfile.txt]
